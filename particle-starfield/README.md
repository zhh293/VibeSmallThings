# 深空觉醒 - 3D粒子星空

一个使用手部追踪控制的交互式3D粒子星空效果。

## 功能特性

- ✨ **3D粒子系统**：15,000个动态粒子形成璀璨星空
- 🤚 **手部追踪控制**：通过摄像头识别手部运动，控制粒子移动
- 🖱️ **鼠标备用模式**：可切换为传统鼠标控制
- 💫 **粒子连接**：距离相近的粒子会形成连接线
- 🎨 **未来科技美学**：赛博朋克风格配色和动画效果
- 📱 **响应式设计**：支持桌面和移动设备

## 使用说明

### 1. 启动页面
- 在浏览器中打开 `index.html`
- 点击"启动量子门"按钮进入星空场景

### 2. 授权摄像头
- 首次访问会请求摄像头权限
- 请允许浏览器访问摄像头

### 3. 手部控制
- 将手放在摄像头前
- 移动**食指尖端**来控制粒子运动
- 右下角会显示摄像头画面和手部骨架追踪
- 绿色状态表示手部已被识别

### 4. 模式切换
- 按键盘 **H** 键可在手部追踪和鼠标模式之间切换
- 手部追踪模式：使用摄像头检测手部
- 鼠标模式：使用传统鼠标控制

## 技术栈

- **Three.js** - 3D图形渲染
- **MediaPipe Hands** - 手部追踪识别
- **HTML5 Canvas** - 手部骨架可视化
- **WebGL** - 硬件加速渲染
- **CSS3** - 动画和视觉效果

## 系统要求

- 现代浏览器（Chrome、Edge、Firefox）
- 支持 WebGL 的设备
- 摄像头（台式机/笔记本内置或外接，手机前后摄像头）
- 良好的光照条件（便于手部识别）

## 性能优化

- 粒子连接仅检查前500个粒子
- 使用 BufferGeometry 提高渲染性能
- 手部追踪使用中等复杂度模型平衡性能和准确度

## 故障排除

### 手部无法识别
1. 确保光线充足
2. 手掌完全展开，面向摄像头
3. 与摄像头保持合适距离（约50-100厘米）
4. 尝试调整手的角度

### 摄像头无法启动
1. 检查浏览器权限设置
2. 确认没有其他应用占用摄像头
3. 尝试刷新页面

### 粒子响应延迟
1. 关闭其他占用CPU的程序
2. 按 H 键切换到鼠标模式
3. 降低浏览器缩放级别
